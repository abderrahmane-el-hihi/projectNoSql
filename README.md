to run the project, you only need to use this command line in your teminal 

```shell
docker comopose up
```

if its not working you should install docker and docker compose and run the command, then open the browser on
```
http://localhost:3000/ 
```

and you will see the frontend page 


in case if you want to shut the project, simply press ctrl + c or run 
```shell
docker compose down
```
